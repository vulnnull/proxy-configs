# Egern 配置

> 基于 [jnlaoshu/MySelf](https://github.com/jnlaoshu/MySelf/tree/main/Egern) 修改，统一使用项目 `rules/` 共享规则。
>
> Egern 是一款新生代网络代理工具，详见 [Egern 官方文档](https://egernapp.com/zh-CN/docs/intro)。

## 使用方式

```
https://raw.githubusercontent.com/vulnnull/proxy-configs/main/egern/egern.yaml
```

## 自定义规则

编辑项目根目录 `rules/` 下的对应 `.list` 文件即可：

- `rules/PrivateDirect.list` — 私人直连
- `rules/PrivateProxy.list` — 私人代理
